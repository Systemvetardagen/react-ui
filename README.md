# Systemvetardagen UI

**Reusable React component library** for Systemvetardagen. 

## Stack

- Typescript
- Tailwind CSS
- ESM and UMD builds

## Installation

```bash
npm install @systemvetardagen/ui
# or
yarn add @systemvetardagen/ui
```
